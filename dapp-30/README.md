# DApps 30

Playing around with Ethereum/Solidity with 30 full stack DApps.

## Local Development

Dependencies:

```bash
npm i -g truffle
npm i -g static-server
```

Dev:

```bash
$backend truffle develop
$truffle migrate --reset
```

```bash
$frontend npm start
```
