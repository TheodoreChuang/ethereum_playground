# ETH Swap

Create our own ERC-20 token (DApp)
Instantly swap between ETH <=> DApp at a fixed rate

## Set Up

Backend / Blockchain:
Run Ganache application (quick start)
`$ truffle migrate --reset`

Client:
`$ npm start`
Import an user from Ganache into MetaMask

### Testing Contracts

Run Ganache application (quick start)
`$ truffle test`
